									Tag
									Usage
									Example
								

								

									<title>
									Assigns a title to the webpage; you can see it on the top of your browser.
									
<title> Condensed Guide to Something </title>

								

								

									<script>
									Used to embed or include a (server-side) script, usually javascript.
									
										
<script type="text/javascript" src="https://code.jquery.com/jquery-1.12.0.min.js"></script>

											
								

								

									<style>
									Used to embed a CSS stylesheet
									
										
<style type="text/css">
	body{
	background-color: lightskyblue;
}
</style>

											
								

								

									<link>
									Used to include external documents, such as stylesheets.
									
										
<link rel="stylesheet" type="text/css" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.6/css/bootstrap.min.css"></link>

											
								

								

									<meta>
									Used to give the browser information about the webpage, such as the language, key words, and text encoding. Meta tags have little effect on the layout of the webpage.
									
										
<meta charset="UTF-8">
<meta name="description" content="HTML, CSS, and Javascript mini guides.">

												Selector
												Usage
												Example (Copy and try it out above!)
											

											

												 p 
												 Generally, a selector with no spaces or special characters will affect every tag of the given type. For example, this selector will affect all paragraph elements.
												
													
												
											

											

												 .colorful 
												 A selector starting with a dot will select all elements with the given class. For instance, this selector will affect the elements with class "colorful" 
												
													
												
											

											

												 #special 
												 A selector starting with a # sign will select the element with the given id. NOTE THAT IDs SHOULD BE UNIQUE. 
												
											

											

												 div h1 
												 A selector with a space, like the one above, will select elements of the second type that are "descendants of" elements of the first type. 
												
											

											

												 div>h1 
												 A selector with a > sign will select elements of the second type that are direct children of elements of the first type.
												
											

											

												 h1+p 
												 A selector with a plus sign will select elements of the second type that immediately follow elements of the first type.
